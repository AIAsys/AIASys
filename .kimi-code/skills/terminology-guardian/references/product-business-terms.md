# 产品商业术语

**覆盖范围**: 产品思维、用户研究、运营策略、商业模式  
**更新日期**: 2026-02-19

---

## 产品核心概念

### 产品市场契合 / PMF

| 字段 | 内容 |
|------|------|
| 标准用词 | 产品市场契合 / PMF |
| 英文 | Product-Market Fit |
| 同义词 | 产品与市场匹配 |
| 禁用词 | 市场契合、产品契合度 |
| 使用场景 | 观点文、项目复盘 |
| 定义 | 产品满足市场需求的程度，用户自发传播的增长状态 |
| 示例句 | 在没找到PMF之前，不要大规模扩张团队。 |
| 备注 | 首次出现可写全称，后续可用PMF |

### 用户旅程 / User Journey

| 字段 | 内容 |
|------|------|
| 标准用词 | 用户旅程 |
| 英文 | User Journey |
| 同义词 | 用户体验路径 |
| 禁用词 | 用户路径、使用流程 |
| 使用场景 | 产品分析 |
| 定义 | 用户从认知产品到完成目标体验的完整过程 |
| 示例句 | 优化用户旅程中的关键节点，可以显著提升转化率。 |
| 备注 | - |

### 用户画像 / Persona

| 字段 | 内容 |
|------|------|
| 标准用词 | 用户画像 |
| 英文 | Persona |
| 同义词 | - |
| 禁用词 | 用户模型、典型用户 |
| 使用场景 | 产品分析 |
| 定义 | 基于调研抽象出的目标用户原型，包含特征、需求、行为模式 |
| 示例句 | 我们的核心用户画像是25-35岁的技术从业者。 |
| 备注 | - |

### 冷启动 / Cold Start

| 字段 | 内容 |
|------|------|
| 标准用词 | 冷启动 |
| 英文 | Cold Start |
| 同义词 | - |
| 禁用词 | 启动期、初期推广 |
| 使用场景 | 项目复盘、产品分析 |
| 定义 | 产品从零开始到获得初始用户和内容的阶段 |
| 示例句 | 社区产品的冷启动最难的是如何产生第一批优质内容。 |
| 备注 | - |

### 网络效应 / Network Effect

| 字段 | 内容 |
|------|------|
| 标准用词 | 网络效应 |
| 英文 | Network Effect |
| 同义词 | 网络外部性 |
| 禁用词 | 规模效应、裂变效应 |
| 使用场景 | 商业分析 |
| 定义 | 产品价值随用户数量增加而提升的现象 |
| 示例句 | 微信的网络效应让它在熟人社交领域难以被替代。 |
| 备注 | - |

## 运营与增长

### 变现 / Monetization

| 字段 | 内容 |
|------|------|
| 标准用词 | 变现 |
| 英文 | Monetization |
| 同义词 | 商业化 |
| 禁用词 | 赚钱、盈利、套现 |
| 使用场景 | 商业分析、项目复盘 |
| 定义 | 将用户价值或流量转化为收入的过程 |
| 示例句 | 工具类产品的变现路径通常比内容产品更清晰。 |
| 备注 | - |

### 留存 / Retention

| 字段 | 内容 |
|------|------|
| 标准用词 | 留存 |
| 英文 | Retention |
| 同义词 | - |
| 禁用词 | 保持、留存率 |
| 使用场景 | 数据分析、产品分析 |
| 定义 | 用户在某时间段后仍然使用产品的比例 |
| 示例句 | 新功能上线后，次周留存提升了15%。 |
| 备注 | 说「留存率」时是指具体指标 |

### 获客成本 / CAC

| 字段 | 内容 |
|------|------|
| 标准用词 | 获客成本 / CAC |
| 英文 | Customer Acquisition Cost |
| 同义词 | - |
| 禁用词 | 拉新成本、推广成本 |
| 使用场景 | 商业分析 |
| 定义 | 获取一个新客户所需的平均成本 |
| 示例句 | 内容获客的优势是CAC极低，但转化周期长。 |
| 备注 | - |

### 生命周期价值 / LTV

| 字段 | 内容 |
|------|------|
| 标准用词 | 生命周期价值 / LTV |
| 英文 | Lifetime Value |
| 同义词 | 用户终身价值 |
| 禁用词 | 长期价值、总价值 |
| 使用场景 | 商业分析 |
| 定义 | 一个用户在产品生命周期内贡献的总价值 |
| 示例句 | 当LTV大于CAC的3倍时，商业模式基本跑通。 |
| 备注 | 常与CAC一起出现 |

## 商业模式

### 订阅制 / Subscription

| 字段 | 内容 |
|------|------|
| 标准用词 | 订阅制 |
| 英文 | Subscription |
| 同义词 | - |
| 禁用词 | 会员制、包月制 |
| 使用场景 | 商业分析 |
| 定义 | 用户定期付费获取产品使用权的商业模式 |
| 示例句 | SaaS产品普遍采用订阅制，保证可预期的收入。 |
| 备注 | - |

### 免费增值 / Freemium

| 字段 | 内容 |
|------|------|
| 标准用词 | 免费增值 |
| 英文 | Freemium |
| 同义词 | 免费+付费 |
| 禁用词 | 免费版、基础版免费 |
| 使用场景 | 商业分析 |
| 定义 | 基础功能免费、高级功能付费的商业模式 |
| 示例句 | Notion采用免费增值模式，个人用户免费，团队协作付费。 |
| 备注 | Freemium = Free + Premium |

### 平台经济 / Platform Economy

| 字段 | 内容 |
|------|------|
| 标准用词 | 平台经济 |
| 英文 | Platform Economy |
| 同义词 | 平台模式 |
| 禁用词 | 平台化、双边市场 |
| 使用场景 | 商业分析 |
| 定义 | 通过平台连接供需双方，从中获取价值的商业模式 |
| 示例句 | 大模型API服务属于平台经济，连接模型提供者和应用开发者。 |
| 备注 | - |
