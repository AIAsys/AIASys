## 使用规范

### 写作时
1. 不确定术语用法时，先查本词汇库
2. 同一篇文章内，术语保持统一
3. 首次出现可中英对照，后续只用标准词

### 迭代时
1. 发现术语不一致，记录到对应文件
2. 新增术语遵循「一个概念一个入口」原则
3. 定期回顾，删除过时术语
