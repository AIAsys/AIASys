## 模型类型

### 大模型 / Large Language Model, LLM

| 字段 | 内容 |
|------|------|
| 标准用词 | 大模型 |
| 英文 | Large Language Model, LLM |
| 同义词 | 大语言模型 |
| 禁用词 | AI大模型、大模型AI、大模型技术、大模型应用 |
| 使用场景 | 通用 |
| 定义 | 参数规模超过10亿，基于Transformer架构的神经网络语言模型 |
| 示例句 | 这篇文章测试了5个主流大模型的代码生成能力。 |
| 备注 | 首次出现可标注LLM，后续只用中文 |
